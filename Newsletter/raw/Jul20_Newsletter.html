July 2020 Newsletter

This is a monthly newsletter of PSL model releases, events, and updates.

User Highlights

PCI-China: Release 2020-Q2 sheds light on China's post-COVID priorities.

Tax-Calculator: A new documentation using Jupyter Book. LINK

Citations: Weifeng Zhong (PCI-China); Max Ghenis and Nate Golden (Tax-Calculator); DeBacker, Evans, and Pomerleau (Tax-Calculator and OG-USA)

ParamTools

A library for parameter processing and validation with a focus on computational modeling projects

Releases and new features

  • Release 0.15.0
    • Provides support for a rich variety of inputs sources from local JSON files to files stored on GitHub, Amazon S3, or Google Cloud Storage (#111, #113).

PCI-China

A library for predicting China's policy changes by mining the text of its state-controlled media

Releases and new features

  • Release 2020-Q2
    • Updates the PCI-China indicator to 2020 Q2, shedding light on China’s post-COVID policy priorities.
General update
  • Using the PCI infrastructure, Weifeng Zhong (Mercatus Center) and his collaborators developed the PCI-Outbreak, an incubating PSL model, that monitors the severity of COVID-19 in China.
  • The PCI-Outbreak was featured in Politico’s China Watcher.

Tax-Brain

An integrator package for multiple open source tax models

Releases and new features

  • Release 2.3.4
    • Offers a patch for building the Conda taxbrain package (#124).

Tax-Calculator

A USA federal individual income and payroll tax microsimulation model

General Updates
  • UBI Center uses Tax-Calculator to compare the effects of extended unemployment relief, universal payments to all or some individuals, and payroll tax cuts.
  • DeBacker, Evans, and Pomerleau's analysis of Joe Biden’s tax plan (using OG-USA and Tax-Calculator) is featured by The Wall Street Journal, Ways & Means Republicans, and The Committee for a Responsible Federal Budget.
  • Tax-Calculator’s documentation gets a makeover using Jupyter Book.
PSL Catalog
  • Behavioral-Responses is a partial-equilibrium behavioral module that works with Tax-Calculator.
  • Cost-of-Capital-Calculator evaluates the effect of US federal taxes on businesses' investment incentives.
  • OG-USA is an overlapping-generations model for evaluating fiscal policy in the United States.
  • ParamTools is a library for parameter processing and validation with a focus on computational modeling projects.
  • PCI-China is an indicator that predicts China's policy changes by mining its state controlled media.
  • Tax-Brain is an integrator package for multiple open source tax models.
  • Tax-Calculator is a USA federal individual income and payroll tax microsimulation model.
  • Tax-Cruncher calculates federal tax liabilities from individual data under a range of policy scenarios.
