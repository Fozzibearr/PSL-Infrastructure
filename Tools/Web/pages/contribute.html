Contributing to PSL

You can contribute to PSL by making a technical contribution to the website, adding a model to the library, or contributing to one of the models already in the library.

The PSL website is hosted on GitHub. When you feel that you have made a valuable contribution, please add it to RELEASES.md. If you think PSL users should know about it, add it to NEWS.md. If your name is not already in contributors.md, add it.

Instructions on how to add a model to the library can be found here, while each model in the library will have its own set of instructions for contributing. You can view each model on the models page for specific instructions on contributing.
